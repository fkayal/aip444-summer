Schema validation checks that input data matches the required structure before the program uses it.

Middleware is code that runs during the request and response cycle. It can log requests, add timing headers, handle Cross-Origin Resource Sharing, or reject invalid input.

Structured Outputs force the model to return data in a specific JSON shape instead of relying on fragile text formatting.

An HTTP API server receives requests from a client and sends responses back using standard web protocols.

A client script can test an API by sending an HTTP POST request with JSON data and printing the server response.
